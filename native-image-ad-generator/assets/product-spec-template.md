# Product Spec Sheet Template

Extract these details from the brand research dossier and compile them before writing NanoBanana prompts. This ensures every prompt describes the product accurately and consistently.

```
PRODUCT SPEC SHEET
==================
Product Name: [Full product name as it appears on the brand's website]

Physical Description: [2-3 sentences describing the physical form — shape,
dimensions, proportions, notable features. Write as if describing to someone
who has never seen the product.]

Color(s): [Detailed color description with hex codes if available from brand
guidelines. Include all visible colors — body, lid, sleeve, interior, etc.
Be specific: "soft sage green with fine speckled texture" not just "green"]

Key Visual Features:
- [Feature 1 — e.g., "speckled matte sage green glaze"]
- [Feature 2 — e.g., "cream silicone slip-on lid with dome top"]
- [Feature 3 — e.g., "no handle — clean modern silhouette"]
- [Feature 4]
- [Feature 5]

Scale Reference: [How big is it relative to common objects? Does it fit
in a car cupholder? Can it be held in one hand?]

Accessories/Components Visible:
- [Component 1 — e.g., "cream silicone lid (slip-on)"]
- [Component 2 — e.g., "sage green silicone sleeve (removable)"]
- [Can be shown with or without certain components? Note that here.]

What It Must NOT Look Like:
- Must NOT [common AI hallucination — e.g., "look like a coffee mug with a handle"]
- Must NOT [competitor confusion — e.g., "look like a stainless steel tumbler"]
- Must NOT [wrong feature — e.g., "have a screw-on lid or flip-top mechanism"]
- Must NOT [wrong finish — e.g., "appear glossy"]
- Must NOT [wrong pattern — e.g., "have dramatic drip-glaze patterns"]
- Must NOT [wrong accessory — e.g., "be shown with a straw or straw hole"]

Brand Colors (from brand guidelines):
- Primary: [Color name] (#HEXCODE)
- Secondary: [Color name] (#HEXCODE)
- Accent: [Color name] (#HEXCODE)
- Background: [Color name] (#HEXCODE)
- Text: [Color name] (#HEXCODE)

Price Point: [Price — this may be needed for offer-heavy or comparison ads]

Key Proof Points (for text overlays):
- [e.g., "4.9 stars from 236 reviews"]
- [e.g., "Handmade in studio"]
- [e.g., "Plastic-free construction"]
- [e.g., "Dishwasher & microwave safe"]
```

## Usage

1. Fill out this template by reading the research dossier
2. Keep this spec sheet accessible while writing NanoBanana prompts
3. Copy-paste the physical description and "Must NOT" sections directly into each prompt
4. Adjust based on concept — some concepts show the product with sleeve, others without; some show it being held, others on a surface
